enum xal::BufferMode

Defines when buffers should be created and how they should be handled.

Enumerator
FULL 

Buffers data upon player creation, keeps results in memory.

ASYNC 

Buffers data upon player creation asynchronously, keeps results in memory.

LAZY 

Buffers when first need arises, keeps results in memory.

MANAGED 

Buffers when first need arises, clears memory after a timeout.

ON_DEMAND 

Buffers when first need arises, clears memory after usage.

STREAMED 

Buffers in streamed mode.
